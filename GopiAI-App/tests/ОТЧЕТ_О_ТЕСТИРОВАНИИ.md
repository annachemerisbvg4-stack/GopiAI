## Как использовать

1. Для сборки и тестирования всего проекта:
   ```powershell
   python build_and_test.py
   ```

2. Для запуска конкретного примера:
   ```powershell
   python run_example.py имя_примера
   ```

3. Для запуска теста:
   ```powershell
   python run_test.py имя_теста
   ```
